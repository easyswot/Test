position

  • position:static; 기본값. 고정
  • position:relative; 현재 위치를 기준으로 변동
  • position:absolute; 상위요소를 기준으로 변동
  • position:fixed; 브라우저를 기준으로 고정
  • position:sticky; relative와 fixed의 믹스

position:sticky; relative와 fixed의 믹스

sticky는 상위요소(body제외)가 있으면 적용 불가 특정 위치에서부터 고정됨.
(z-index값을 크게 지정해서 겹친 다른 요소보다 항상 위로 올라오도록)

1. position:static; 기본값.

position:static; left:800; 위치변화 없음

2. position:relative; 현재 위치를 기준으로 변동

position:relative;
left: 800px;
position:relative;
top: 250px;
position:relative;
right: 200px;
position:relative;
bottom: 200px;
position:relative;
left: 20em; bottom: 3em;
position:relative;
left: 100em; top:4em;

3. position:absolute; 상위요소를 기준으로 변동

기준이 될 상위요소는 position:relative로 설정 [z-index:n;] z-index의 값이 클수록 겹쳤을 때 위로 올라옴 (z-index는 태그 입력 시 자동으로 1씩 증가) 음수값도 가능.

position:absolute;
left: 800px;
position:absolute;
right: 200px;
position:absolute;
top: 10px;
position:absolute;
bottom: 1em;
position:absolute;
right: 0; bottom: 0;
position:absolute;
left: 50%; top: 50%;
