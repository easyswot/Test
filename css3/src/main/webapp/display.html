box-model

  1. inline요소 - 한 줄에 나란히 오면서 상하 margin, 크기 적용 불가 <a>, <b>, <label>, <span>, ...
  2. block요소 - 한 줄에 나란히 올 수 없고
    margin, 크기 적용 가능 <p>, <hn>, <section>, <div>, ...

display 요소가 화면에 나타나는 방식 지정

  • display: inline; block인 요소를 inline으로 바꿈
  • display: block; inline인 요소를 inlineblock으로 바꿈
  • display: inline-block; inline과 block을 합친 성격으로 바꿈
  • display: none; 화면에서 요소를 사라지게 함

display:inline; block인 요소를 inline으로

block인 p

block인 div
block인 aside

display:block; inline인 요소를 block으로

inline인 a inline인 b inline인 span

display:inline; inline과 block을 합친 성격

block인 p

block인 div
block인 aside
inline인 a inline인 b inline인 span

display:none; (영역까지) 화면에서 요소를 사라지게

block인 p

block인 aside
