반응형 기술 - Media queries

[구문 작성법] @media 장치종류 and (조건) { 선택자 {속성: 속성값;} }

Media queries의 장치종류

  • all = 장치 종류 무관
  • screen = 모니터로 확인하는 모든 컴퓨터(스마트폰, 테블릿 포함)
  • print = 프린터
  • speech = 시각장애인용 음성낭독기

Media queries의 조건들

  • width and height of the viewport = 뷰포트의 너비나 높이
  • width and height of the device = 장치의 너비나 높이
  • orientation(landscape or portrait) = 화면 방향(가로/세로)
  • resolution = 해상도
