margin

요소의 외부(테두리 기준) 여백

margin의 방향별 지정 속성

  • margin-top:n; 외부 위쪽에만 n적용
  • margin-right:n; 외부 오른쪽에만 n적용
  • margin-bottom:n; 외부 아래쪽에만 n적용
  • margin-left:n; 외부 왼쪽에만 n적용

margin의 값 개수별 적용법

  • margin:n; 양사방 동시에 n적용
  • margin:n n; 상하, 좌우가 서로 같은 값
  • margin:n n n; 상, 좌우 동일, 하의 순서
  • margin:n n n n; 상, 우, 하, 좌의 순서

항상 요소가 수평 중앙에 위치

= 좌우 margin을 항상 같게 함. margin-left: auto; margin-right: auto; 또는 margin:n auto;

inline요소 margin

inline(한 줄에 나란히 오는) 요소는 상하 margin은 적용 불가, 좌우 margin은 인접요소마다 각각 적용
